# 🔓 totp-js

No dependency TOTP generation library for both the browser and node.

## ✨ Usage

```
// esm
import TOTP from 'totp-js'
TOTP.generateTOTP(process.env.2FA_SEED)

// cjs
const { generateTOTP } = require('totp-js')
generateTOTP(process.env.2FA_SEED)
```
